# 澳洲大件家具独立站 - 最终执行版 PRD (V2.0 锁定版)

## 1. 项目最高指令 (Project North Star)

* **终极目标**：**1:1 像素级全功能复刻澳洲本土标杆 `https://mcphails.com.au/**`。从前端视觉交互、购物车逻辑，到后端的运费计算、支付分期，做到功能体验完全一致。
* **开发资源**：单人全栈开发（1名）。
* **交付周期**：极限交付，总工期 1 个月（4 周）。

## 2. 基础设施与云资源配置 (已锁定：2,500 元/年)

*(采用“单服务器全栈部署 + 静态资源分离”的高性价比强力架构)*

* **核心服务器（1,100 元）**：腾讯云新加坡通用型（2核 8G内存，120GB系统盘，3.5TB 免费月流量包）。承载 Nuxt 3 前端渲染、Medusa 后端引擎及 PostgreSQL/Redis 数据库。
* **静态资源池（500 元）**：对象存储 OSS（200GB 存储 + 20GB 下行流量），接管全站海量高清沙发大图。
* **商用通信接口（500 元）**：SendGrid 商业邮件 API，确保订单、发货通知精准送达澳洲邮箱。
* **基础网络资产（200 元）**：国际域名续费与 SSL 安全证书。

## 3. 对标 McPhails 的核心功能清单 (1:1 复刻清单)

*为了保证在 1 个月内完工，我们将 McPhails 的核心功能彻底拆解，严格按此清单开发，绝不节外生枝：*

### 🟢 官网交互与展现 (Frontend UI)

* [ ] **巨型导航 (Mega Menu)**：完美复刻 McPhails 顶部的下拉分类矩阵。
* [ ] **多规格沙发展示 (PDP)**：复刻其商品详情页逻辑，点击不同材质/颜色色块，主图瞬间切换，且绝不出现“无库存的幽灵规格”。
* [ ] **图片懒加载与放大镜**：接入 OSS 动态压缩，实现商品主图丝滑滑动与局部放大功能。

### 🟡 交易转化漏斗 (Cart & Checkout)

* [ ] **右侧全局滑出式购物车 (Drawer Cart)**：点击加购后不跳转，右侧丝滑拉出购物车面板。
* [ ] **免注册游客结账 (Guest Checkout)**：复刻其极简流程，只需填邮箱、地址即可付钱。
* [ ] **Google 一键登录**：在结账页顶部植入，加速回头客下单。

### 🔴 澳洲本地化商业规则 (Business Logic)

* [ ] **一口价与偏远拦截 (Shipping Rules)**：根据澳洲邮编，实时判定是 $59 一口价，还是直接置灰拦截“偏远地区暂不支持配送”。
* [ ] **GST 消费税强制含税价**：全站商品价格全部显示为含 10% GST 的最终澳元 (AUD) 价格。

### 🟣 跨境支付网关 (Payments)

* [ ] **基础信用卡收单**：对接 Stripe，支持 Visa, Mastercard 等无缝嵌在网页内扣款。
* [ ] **澳洲“四大分期”复刻**：必须在支付页亮出 **Afterpay / Zip** 等先买后付的图标和选项（通过 Stripe 聚合接口实现）。
